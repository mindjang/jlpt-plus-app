# 사용자 앱 화면 개선 사항

## 🔍 분석 결과

### 현재 구현된 기능
1. ✅ 홈 페이지 - 기본 학습 시작 화면
2. ✅ 도서관 - 레벨별 콘텐츠 선택
3. ✅ 학습 세션 - SRS 기반 학습
4. ✅ 퀴즈 시스템 - 경험치, 레벨업, 배지
5. ✅ 통계 페이지 - 기간별/학습별 통계
6. ✅ 마이 페이지 - 설정, 멤버십 관리

### 🚨 발견된 부족한 기능

#### 1. 홈 페이지 개선 필요
**현재 문제:**
- `StreakChip`이 항상 `count={0}`으로 하드코딩됨
- 실제 streak 데이터를 가져오지 않음
- 오늘의 학습 목표 진행도 표시 없음
- 오늘의 학습 요약 정보 없음

**개선 사항:**
- [ ] 실제 streak 데이터 로드 및 표시 (`getStreak` 사용)
- [ ] 오늘의 학습 목표 진행도 표시 (일일 목표 대비 완료율)
- [ ] 오늘의 학습 요약 카드 추가 (학습한 카드 수, 정답률, 학습 시간)
- [ ] 최근 학습한 내용 미리보기

#### 2. 학습 히스토리/기록 페이지 부재
**필요한 기능:**
- [ ] 학습 세션 히스토리 페이지
- [ ] 날짜별 학습 기록 조회
- [ ] 학습한 단어/한자 목록
- [ ] 복습 예정 카드 목록

#### 3. 즐겨찾기/북마크 기능 부재
**필요한 기능:**
- [ ] 단어/한자 즐겨찾기 추가/제거
- [ ] 즐겨찾기 목록 페이지
- [ ] 빠른 접근을 위한 북마크 기능

#### 4. 학습 알림/리마인더 부재
**필요한 기능:**
- [ ] 복습 알림 (복습 예정 카드가 있을 때)
- [ ] 일일 학습 목표 달성 알림
- [ ] 연속 학습 일수 유지 알림

#### 5. 성취/배지 표시 개선
**현재 상태:**
- 퀴즈 시스템에는 배지가 있지만 홈에서 표시되지 않음

**개선 사항:**
- [ ] 홈 페이지에 최근 획득한 배지 표시
- [ ] 배지 갤러리 링크 추가
- [ ] 성취 진행도 표시

#### 6. 학습 목표 설정 및 추적
**필요한 기능:**
- [ ] 주간/월간 학습 목표 설정
- [ ] 목표 달성 진행도 표시
- [ ] 목표 달성 축하 메시지

#### 7. 검색 기능 개선
**현재 상태:**
- 도서관에 검색 기능이 있지만 개선 여지 있음

**개선 사항:**
- [ ] 최근 검색어 저장
- [ ] 인기 검색어 표시
- [ ] 검색 필터 (레벨, 타입 등)

#### 8. 학습 통계 시각화 개선
**현재 상태:**
- 기본적인 통계는 있지만 시각화가 부족

**개선 사항:**
- [ ] 차트/그래프 추가 (학습 추이, 정답률 등)
- [ ] 히트맵 개선
- [ ] 레벨별 상세 통계

#### 9. 오프라인 학습 지원
**필요한 기능:**
- [ ] 오프라인 모드 지원
- [ ] 학습 데이터 로컬 캐싱
- [ ] 오프라인 학습 기록 동기화

#### 10. 소셜 기능 (선택적)
**선택적 기능:**
- [ ] 친구 추가/초대
- [ ] 학습 랭킹
- [ ] 학습 공유

## 📋 우선순위별 개선 계획

### 🔴 높은 우선순위 (즉시 개선 필요)
1. **홈 페이지 streak 데이터 표시** - 사용자 동기 부여에 중요
2. **오늘의 학습 요약 카드** - 사용자가 하루 학습을 한눈에 파악
3. **학습 히스토리 페이지** - 학습 기록 조회 기능

### 🟡 중간 우선순위 (사용자 경험 개선)
4. **즐겨찾기 기능** - 자주 찾는 콘텐츠 빠른 접근
5. **학습 목표 설정 및 추적** - 목표 의식 강화
6. **성취/배지 홈 표시** - 동기 부여

### 🟢 낮은 우선순위 (추가 기능)
7. **학습 알림/리마인더** - 푸시 알림 구현 필요
8. **검색 기능 개선** - 현재 기능으로도 충분
9. **통계 시각화 개선** - 현재도 기본 기능 제공
10. **오프라인 지원** - 기술적 복잡도 높음

## 🎯 즉시 구현 가능한 개선 사항

### 1. 홈 페이지 streak 데이터 표시
```typescript
// app/(root)/home/page.tsx 수정 필요
import { getStreak } from '@/lib/firebase/firestore/dailyActivity'
// StreakChip에 실제 데이터 전달
```

### 2. 오늘의 학습 요약 카드
```typescript
// getDailyActivity 사용하여 오늘의 학습 데이터 표시
import { getDailyActivity } from '@/lib/firebase/firestore/dailyActivity'
```

### 3. 학습 히스토리 페이지
```typescript
// 새 페이지: app/(root)/history/page.tsx
// getRangeActivity 사용하여 히스토리 조회
```
